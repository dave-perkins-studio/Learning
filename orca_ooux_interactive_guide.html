Vocabulary

ORCA Glossary

Object

The "nouns" of your system. Persistent entities that users interact with and care about.

Relationship

How objects connect to one another, defined by direction and cardinality (e.g., 1-to-many).

CTA

Calls to Action (the "verbs"). Actions that users or the system perform on specific objects.

Attribute

The "adjectives" or data points. Information that describes or defines an object's state.

Nested Object

A relationship where one object is visually or logically contained within a parent view.

Force Rank

A strict prioritization method requiring a unique rank for every item—no ties allowed.
